The Problem

Ideal: Property owners/stakeholders can understand the current status of the shoreline of Lake Simcoe, as defined by the MNRF in 2013.

Reality: Lake Simcoe shoreline categorization is currently unavailable to public users.

Proposal: Create a web application that can be used by property owners and municipal groups to understand and view current shoreline classification so that they make informed decisions on how to manage shoreline activities in the future.

Technology Used: ArcGIS Experience Builder, ArcGIS Hub

There is currently no accessible platform for the public to view and understand information about the Lake Simcoe shoreline classification project. The goal of this project was to create an easy-to-access resource for cottagers, property owners, and organizations to easily view shoreline classifications (via the Experience Builder app) and learn more about the value of shoreline classification and what is being done with that information (via the ArcGIS Hub site).
